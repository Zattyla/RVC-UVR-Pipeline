# 🌊 K-Studio AI (v2.0)
An all-in-one professional environment for AI Cover production, featuring high-fidelity stem separation and RVC voice conversion.

## 🛠️ Features
- **Stem Separation:** Powered by `audio-separator` with Roformer & MDX-Net support.
- **Voice Conversion:** Integrated RVC (Retrieval-based Voice Conversion) with RMVPE pitch extraction.
- **Model Management:** In-app downloader for `.zip` models (GDrive/Direct links).
- **UI:** Custom Blue Pastel Gradio interface with easy download buttons.

## 🚀 Quick Start
1. Open in **Google Colab**.
2. Enable **T4 GPU**.
3. Run **Install Dependencies**.
4. Launch the **Gradio Interface** and start creating.
